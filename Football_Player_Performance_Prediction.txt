Football Player Performance Prediction
 https://football-player-perfm-pred.herokuapp.com/
 
 This is a project that is built using the Python and Flask Framework. This web app can predict the overall score of a football player given various attributes and match information. 
 This webapp is deployed on the Heroku Platform.
 
 Reading data from a SQLITE database. 
 i) The database consists of records of players performance in a match. Converting the match records into player records(i.e grouping the records based on player id using Clean_and_Merge.py and Clean_and_Merge_Target.py)
 ii) Creating Pipeline( numerical and numerical)
 iii) Modeling
